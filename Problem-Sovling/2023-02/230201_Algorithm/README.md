# 230201-BOJ-행렬 곱셈 순서

## 낚시왕

- https://www.acmicpc.net/problem/11049

- 행렬곱과 `DP`를 활용한 문제

#### 풀이과정

1. `DP`는 항상 답을 모른다/안다 로 나뉘는거 같다 ㅠ

2. n*n 짜리 행렬을 두고 `DP`로 풀이한다.

3. `(i,j)`의 값은 `i`번째 자리 부터 `j`번째 자리의 계산의 최소비용

4. 대각선으로 n*n행렬을 계산하면 부분적으로 주어진 행렬들의 곱셈 순서의 값을 구할 수 있다.
