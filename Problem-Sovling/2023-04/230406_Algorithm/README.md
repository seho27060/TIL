# 230406-BOJ-미로 탐색

## 미로 탐색

- https://www.acmicpc.net/problem/2178

- 주어진 미로에서 (1,1) 부터 (n,m)까지 최소 도착 거리를 출력하라.

### 풀이과정

1. `BFS`로 탐색, 

2. 입력이 문자열이 "붙여서" 입력됨. 
   
   - 입력되면 파싱하여 `board`를 생성

3. `board`를 이동하면서 `check`이라는 이중 배열에 이동 거리를 갱신한다.
