#### Topic 30 변환 프로그래밍

- 프로그램이란...단순히 보면 입력에 대한 출력.

- 객체 지향 프로그래밍에서 자주 볼 수 있는 연쇄적 호출
  
  ```java
  method()
      .find()
      .calculate()
      .aggregate()
  ```
  
  보다는..
  
  ```java
  a = method();
  b = find(a);
  c = caculate(b);
  d = aggregate(c);
  ```
  
  와 같은 방법이 프로그래밍의 변환에 적합하며, 코드내 결합도를 낮추기 적절하다.
