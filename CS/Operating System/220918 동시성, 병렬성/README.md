# 동시성, 병렬성

## 동시성(Concurrency) vs 병렬성(Parallelism)

| 동시성                                        | 병렬성                                   |
| ------------------------------------------ | ------------------------------------- |
| 동시에 실행되는 것 같이 보이는 것                        | 실제로 동시에 여러 작업이 처리되는 것                 |
| 싱글코어, 멀티코어에서 멀티 쓰레드(Multi Thread)를 동작하는 방식 | 멀티 코어에서 멀티 쓰레드(Multi Thread)를 동작하는 방식 |
| 한번에 많은 것을 처리                               | 한번에 많은 일을 처리                          |
| 논리적인 개념                                    | 물리적인 개념                               |
